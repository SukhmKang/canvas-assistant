• What data does Canvas Assistant Collect?
  • 1
    Canvas Assistant handles the user's Canvas API Token and their university name.
  • 2
    None of this data is stored or collected.
  • 3
    Canvas Assistant does not collect any information from the user's web pages or browsing activity. The only information that is handled is what the user inputs in the form.
  • How does Canvas Assistant use that data?
  • 1
    Using the provided data, Canvas Assistant accesses the user's Canvas course data, assignments, and a list of students in the user's courses.
  • 2
    Even though the user never provides Canvas Assistant with their name, using the API token can access that information, however it is never saved or shared.
  • How does Canvas Assistant disclose/store your data?
  • 1
    Information is never disclosed, saved, or shared to anybody except the user themselves.
